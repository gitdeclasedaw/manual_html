Título

La etiqueta <title> define el título de un documento.
Esta etiqueta es obligatoria dentro de la sección head.

La etiqueta <title> se muestra en:

  • La barra superior del navegador.
  • Cuando lo agregamos a nuestros favoritos.
  • Los tabs de los nuevos navegadores.
  • Los resultados de los motores de busqueda.

Un ejemplo claro seria el siguiente:


<!doctype html>
 
<html lang=“es”>
  
  <head>
  
  		<title>Mi primer documento HTML - Manual para lenguaje de marcas</title>
  
  </head>
  
  	<body>
  
  		Este es el texto de mi primer documento.
  
  	</body>
  
</html>
