Nav (Navegación)

La directiva <nav> es una directiva nueva en HTML es utilizada como contenedor para englobar elementos de navegación en un documento HTML. Los elementos de navegación pueden ser: el menu principal de navegacion, el menu lateral de navegacion, etc... Su introducción en HTML se debe a que existian demasiadas directivas <div> en los documentos HTML. Su presencia delimita las areas de navegacion de la pagina web, proporcionando una mejor vista del código HTML.

Aquí tenemos un ejemplo de un posible uso:


<!DOCTYPE HTML>
 
<html>
  
	<head>
  
		<title>Título del documento - Manual para lenguaje de marcas</title>
  
	</head>
  
	<body>
  
		<nav>
 
			<ul>
  
				<li><a href="1.html">menu 1</a></li>
  
				<li><a href="2.html">menu 2</a></li>
  
				<li><a href="3.html">menu 3</a></li>
  
			</ul>
  
		</nav>
  
		<p>La directiva <nav> es una directiva nueva en Html es utilizada como contenedor
		para englobar elementos de navegación en un documento html.</p>
 
	</body>
  
</html>
