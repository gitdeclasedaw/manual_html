Hyperenlace Interno

Para crear un enlace de este tipo es necesario dos componentes, que para aclararnos los vamos a nombrar de la siguiente forma:

  1. El enlace: Sería el link, lo que aparecerá en la página para que el usuario haga clic. Sería el enlace de origen propiamente dicho.

  2. El ancla: Además se requiere una marca, para saber dónde se dirige el enlace. Es el destino donde nos llevará el navegador al pulsar el link. Le llamamos ancla porque nos permite anclar a esa posición otros enlaces.

Ambos elementos se crean con la misma etiqueta <a>, tanto el enlace como el ancla. Solo que usaremos distintos atributos dentro de esa etiqueta. Por ejemplo:


<!DOCTYPE html>

<html>

	<head>

		<title>Título del documento - Manual para lenguaje de marcas</title>

	</head>

	<body>

		<a href="#nombrerelacion">Texto del enlace</a>
	
		....
		....
		DOCUMENTO
		....
		....
	
		<a name="nombrerelacion">PARTE DEL DOCUMENTO DONDE LLEGAMOS
		CON EL ENLACE DE ARRIBA</a>
	
		....
		....
		DOCUMENTO
		....
		....

	</body>

</html>

También podemos usar enlaces internos como "#top" que nos lleva al inicio de la página sin necesidad de ancla.
