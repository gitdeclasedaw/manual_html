body (Cuerpo)

Con las directiva <body>, se delimita el cuerpo del documento, que es el contenedor para todos los elementos visibles para el usuario, como párrafos, imágenes, listas, etc.

<!DOCTYPE html>
	
<html>

	<head>
	
		<title>Título del documento - Manual para lenguaje de marcas</title>
		
	</head>
	
<body>

	<p>...cuerpo del documento...</p>
	
</body>

</html>
