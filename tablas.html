Tablas

Las tablas más sencillas de HTML se definen con tres etiquetas: <table> para crear la tabla, <tr> para crear cada fila y <td> para crear cada columna.

El uso de cada una de estas seria <tr> para las filas y <td> para las columnas, estando esta ultima encerrada en <tr> y todas encerradas en <table>. Tambien podemos añadir

A continuación se muestra el código HTML de una tabla sencilla:


	<table>
		<tr>
			<td>Fila 1 en columna 1></td>
			<td>Fila 1 en columna 2></td>

		</tr>
		<tr>

			<td>Fila 2 en columna 1></td>
			<td>Fila 2 en columna 2></td>

		</tr>

	</table>

Tambien podemos añadir el atributo border a <table> para mostrar el borde de la tabla y usar rowspan y colspan en <td> para expandir una celda haciendo que se una con otra que tenga a cualquierda de sus 4 lados y en adelante a esa misma dirección elegida, habiendo que elimnar las correspondientes celdas a las que se va a unir. Un ejemplo sencillo sería el siguiente en una tabla de 5X4:


<table border>
	<tr>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
	</tr> 
	<tr>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
	</tr>
	<tr>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
		<td>celda</td>
	</tr>
	<tr>
		<td>celda</td>
		<td>celda</td>
		<td colspan=2 rowspan=2>CELDAS (4,3),(4,4),(5,3),(5,4) UNIDAS</td>
	</tr>
	<tr>
		<td>celda</td>
		<td>celda</td>
	</tr> 
</table>
