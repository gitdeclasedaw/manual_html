Texto Preformateado

<pre> y </pre> se utiliza para mostrar texto preformateado, de forma que lo que todo el texto que aparece roadeado por la directiva <pre> aparecerá tal y como se ponga.

Los atributos globales mano cursor y los eventos HTML5 mano cursor, pueden acompañar la directiva <pre>


<!DOCTYPE html>

<html>

	<head>

		<title>Título del documento - Manual para lenguaje de marcas</title>

	</head>

	<body>

		<pre>

		Este texto aparecerá

		escrito en el navegador con este mismo formato

		utilizando un tipo de letra monoespaciada.

		</pre>

	</body>

</html>
