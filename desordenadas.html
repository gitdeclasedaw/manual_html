Listas Desordenadas

Las listas no ordenadas (Desordenadas) son las más sencillas y las que más se utilizan. Una lista no ordenada es un conjunto de elementos relacionados entre sí pero para los que no se indica un orden o secuencia determinados. La etiqueta <ul> encierra todos los elementos de la lista y la etiqueta <li> cada uno de sus elementos.

<!DOCTYPE html>
		
<html>

	<head>

		<title>Título del documento - Manual para lenguaje de marcas</title>

	</head>

	<body>

		<h1>Manual HTML DAW1</h1>

		<ul>
	
			<li>Inicio</li>
	
			<li>Noticias</li>
	
			<li>Artículos</li>
	
			<li>Contacto</li>

		</ul>

	</body>

</html>
